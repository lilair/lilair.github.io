Linux基础命令

​ cd /root 切换到root文件夹

​ cd ~ 进入到家目录

​ ls -a 列出目录所有文件，包括隐藏的文件

​ ls -A 列出非隐藏文件夹

​ ls -r 倒序排列

​ ls -h 以M显示文件的大小 易读

​ ls -l 显示文件的权限大小等详细的信息

​ ls -a* 通配符显示以a开头的文件和文件夹

​ pwd

​ mkdir /root/a 在root文件夹下面建立a文件夹

​ mkdir -p root/c/a 在给定目录下面创建文件夹 如果中间目录不存在，就创建

​ a.显示文件 b.合并文件 c.创建文件

​ cat 文件路径/文件

​ cat >文件

​ cat 文件1 文件2>/>> 文件0

​ rm [选项] 文件

​ rm -rf 文档 不询问

​ rmdir -p a/b/c 从目录中删除一个或者多个子目录

​ 复制命令，将源文件（多个文件）复制到目标目录

​ -i 提示 -r复制目录以及目录内所有项目 -a保持时间一致

​ cp -ai a.txt root

​ 剪切，重命名

​ mv 1.txt 2.txt

​ mv 1.txt 2.txt /root

​ 浏览文件

​ less 1.txt

​ 分页显示文件，不覆盖窗口

​ +n -n

​ more +3 1.txt
